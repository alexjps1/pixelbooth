Line 4 is for $PASSWORD (login password for accessing remote server via SSH)

Line 6 is for $PIXELATION (3 integer arguments, separated by spaces, for pixelate.py)
Line 7 is for $SOCIAL (social media page for post consent)
Line 8 is for $MODE (can be "mosaic" or "overlay")
Line 9 is for $DIMENSIONS (x and y dimensions for size of overlay output image eseparated by space)
